Changes (0.5.1)
---------------
1. Обновлены все иконки компонентов.
2. Добавлен пункт меню у корневого элемента проекта - запуск режима отладки. 
   На отладку запускается модуль run.py (точка входа в проект) 
3. Обновлен плагин к ide Editra до версии 0.5.1 - изменена функция запуска отладчика
   - теперь отладчик запускается через модуль ic/Scripts/debug.py
4. Теперь при вызове корня проекта на редактирование открывается файл запуска проекта run.py, а не __init__.py 
5. У компонента GridDataset изменен рендер для текстовых значений - теперь текст не заезжает на 
   пустые ячейки
6. Каждый компонент тепеперь может иметь свою собственную панель инструментов (верхнюю) 
   в графическом редакторе (управляется через функцию GetToolPanelCls).
7. Добавлен интерфейс класса управления ресурсом в редакторе ресурсов. В дальнейшем на этот
   класс будут переложены все функции по управлению ресурсом в редакторе ресурсов.
8. Введен базовый класс для дизайнеров icdesignerinterface.icDesignerInterface
9. Обновлен редактор форм (фон).
10. Обновлена панель инструментов: все компоненты теперь в линейном списке с подписями типов объектов.
11. Новые картинки для редактора ресурсов.
12. Если модуль ресурса не открыт, по двойному щелчку в редакторе атрибута редактора ресурса, 
    теперь модуль ресурса открывается на редактирование
